# Global Superstore Interactive Dashboard

## Project Overview

This project is an interactive business intelligence dashboard built with Streamlit using the Global Superstore dataset. The dashboard allows users to explore sales performance, profit, customer information, and regional trends through interactive filters and visualizations.

## Dataset

* **Dataset:** `Global_Superstore.csv`

## Objectives

* Clean and prepare the dataset.
* Build an interactive Streamlit dashboard.
* Analyze sales and profit performance.
* Display key performance indicators (KPIs).
* Provide business insights using interactive charts.

## Dashboard Features

* Region Filter
* Category Filter
* Sub-Category Filter
* Total Sales KPI
* Total Profit KPI
* Top 5 Customers by Sales
* Sales by Category
* Profit by Region
* Sales by Segment
* Monthly Sales Trend
* Interactive Data Table

## Technologies Used

* Python
* Pandas
* Plotly
* Streamlit

## Project Structure

* `app.py`
* `Global_Superstore.csv`
* `requirements.txt`
* `README.md`

## How to Run

1. Install the required libraries:

```bash
pip install -r requirements.txt
```

2. Run the Streamlit application:

```bash
streamlit run app.py
```

## Results

The dashboard provides interactive visualizations that help users analyze business performance, identify sales trends, compare regional profit, and monitor customer activity through dynamic filters.

## Conclusion

This project demonstrates practical skills in business intelligence, dashboard development, data visualization, and interactive analytics using Python and Streamlit.
